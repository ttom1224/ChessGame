# ChessGame
使用Java AWT開發的一個1V1的西洋棋小遊戲，選定角色會顯示出它可以移動的位置，遊戲會以國王被吃掉後的總分來判斷輸贏，點擊主畫面上的Rule會顯示出得分規則

## Features
選定角色後左方會跑出對應角色的SDG標的，被吃掉的角色會照陣營顯示於右方
